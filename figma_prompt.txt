DESIGN BRIEF — "LMS" web app for a learning center. UI language: UZBEK (Latin). Use every label text EXACTLY as written below, do not translate to English.

=== GENERAL RULES ===
- Desktop web app, 1440px wide, light theme, flat and clean admin-dashboard style. No gradients, no shadows heavier than subtle.
- Colors: primary navy #1E2A4A, accent blue #2F57C7, success green #2E9E5B, danger red #D94841, warning amber #E8A13C, page background #F5F6F8, cards white with 1px #E2E5EC border, border radius 10px.
- Font: Inter. Body 14-15px, page titles 22px semibold.
- APP SHELL on every screen except Login: dark navy top navbar. Left: logo text "LMS". Menu: "Bosh sahifa", "Kurslar", "Guruhlar", "Jadval", "Normativ", "Do'kon", "Yangiliklar". Right: small avatar + user name "Aziz Karimov" + button "Chiqish". Navbar itself carries a tiny chip "B2.1".
- MOST IMPORTANT — TASK BADGES: this design is a teaching artifact for students. Every page header shows a small chip on the top-right with the page's task ID, e.g. "B3.1". Every button, form or block that belongs to a DIFFERENT task gets its own tiny chip right next to it, e.g. the buy button gets chip "B8.4". Chip style: background #EEF1F7, text #2F4070, 11px, radius 10px, padding 2px 8px. Never omit these chips.
- Buttons: primary = filled blue, secondary = white with border, danger = red. Button labels are short nouns/actions: "Saqlash", "Qo'shish", "Bekor qilish", "O'chirish".
