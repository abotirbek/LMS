Loyiha maqsadi

Nima quramiz va qanday ishlaymiz

Nima quramiz

O'quv markaz uchun LMS: kurslar, guruhlar, darslar jadvali, davomat, normativ (dars savollari), coin tizimi va yangiliklar. Django template + forms, hech qanday tashqi frontend framework yo'q.

Qanday ishlaymiz

Loyiha 9 bosqich, 45 taskka bo'lingan. Task olasiz, kartadagi vazifani o'qiysiz, qabul mezonlari to'liq bajarilgandagina task yopiladi. Har bosqich boshida sahifalarning HTML mockup dizayni beriladi.

Texnologiya

Python 3.14, Django 6.0, SQLite, Pillow. Loyiha tayyor: lms/ papkasi, 7 ta app, 18 ta model, migratsiyalar qilingan. Sizga faqat view, form, template va admin qismi qoldi.

Qoida: kod yozishdan oldin tegishli modelni oching va fieldlarini o'qing. Taxmin qilmang — model nomlari, related_name lar va choices qiymatlari quyidagi ma'lumotnomada aniq yozilgan.

role_required dekoratori

Vazifa:

accounts/decorators.py da role_required(*roles) dekoratorini yozing: login bo'lmagan userni login ga yo'naltiradi, roli mos kelmaganga 403 qaytaradi. Keyingi barcha bosqichlarda shu dekorator ishlatiladi.

Qabul mezoni:
  • @role_required('teacher') qo'yilgan viewga student kirsa 403 oladi
  • Login qilmagan user login sahifasiga otadi
  • Bir nechta rol berish mumkin: @role_required('teacher', 'admin')
from functools import wraps
from django.http import HttpResponseForbidden
from django.shortcuts import redirect

def role_required(*roles):
    def decorator(view_func):
        @wraps(view_func)
        def wrapper(request, *args, **kwargs):
            if not request.user.is_authenticated:
                return redirect('login')
            if request.user.role not in roles:
                return HttpResponseForbidden("Ruxsat yo'q")
            return view_func(request, *args, **kwargs)
        return wrapper
    return decorator

class CourseForm(forms.ModelForm):
    class Meta:
        model = Course
        fields = ['title', 'description', 'duration_months', 'is_active']

    def clean_duration_months(self):
        value = self.cleaned_data['duration_months']
        if not 1 <= value <= 24:
            raise forms.ValidationError("Davomiylik 1-24 oy oralig'ida bo'lishi kerak")
        return value
